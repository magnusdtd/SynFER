\subsubsection{Tổng hợp biểu cảm khuôn mặt}
Phân tích 3 phương pháp tiếp cận kinh điển:
\begin{itemize}
    \item Dựa trên mô hình vật lý cơ-xương (Physically Based).  
    \item Dựa trên biến đổi hình thái hình học (Morph-Based).  
    \item Ánh xạ biểu cảm hiệu năng (Expression Mapping/ERI).
\end{itemize}
